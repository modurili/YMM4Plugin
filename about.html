YMM4 Plugin Catalog について

ゆっくりMovieMaker4 のプラグインを探しやすくするためのカタログサイトです。

📦 このサイトについて

YMM4 Plugin Catalog は、GitHub 上で公開されている YMM4（ゆっくりMovieMaker4）向けプラグインを 自動的に収集し、カテゴリ別に整理して一覧表示するカタログサイトです。

プラグインの検索、カテゴリフィルタリング、並べ替え機能を提供し、 ユーザーが目的のプラグインを素早く見つけられるようにすることを目指しています。

🔄 データの収集方法

以下の GitHub トピックが付けられたリポジトリを自動的に検索・収集しています：

  • ymm4-plugin
  • ymm-plugin
  • YMM4Plugin

プラグイン開発者の方は、ご自身のリポジトリに上記いずれかのトピックを追加すると、 自動的にこのカタログに掲載されます。
追記：YMM4が推奨しているのはymm4-pluginです。
参考：リポジトリのトピックについて
